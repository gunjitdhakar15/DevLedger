# DevLedger 🗂️

**DevLedger** is a team-focused project dashboard that helps developers and teams track their progress, assign tasks, and get weekly summaries.

## 🚀 Tech Stack
- React.js
- TailwindCSS
- Vite

## 📌 Features (Planned)
- Team registration & login
- Task assignments
- Weekly summaries
- Realtime status updates
